# 3.3.4 プロセッサの性能
クロック周波数

クロック：コンピュータが動作のタイミングを計るための一定の周波数の信号を出力する装置

クロック周波数が高いほど、CPUの動作速度は速く、命令実行速度は速い

命令実行時間はクロックサイクル時間とCPIで求められる
